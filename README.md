Result Analysis System

A web-based Result Analysis System built using React (Vite) and Firebase that helps manage, analyze, and visualize student performance efficiently.

🚀 Features
📌 Upload and manage student results
📊 Automatic grade calculation system
📈 Performance analysis and insights
📉 Attendance tracking and percentage calculation
🔐 Firebase authentication & database integration
⚡ Fast and responsive UI with Vite + React
